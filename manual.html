TallerPRO360 ERP

Manual de Usuario

Este manual explica cómo utilizar las funciones principales del ERP TallerPRO360. Incluye gestión de clientes, inventario, órdenes de trabajo y generación de facturas.

1. Gestión de Clientes

  • Para agregar un cliente, ve al módulo Clientes.
  • Ingresa nombre, teléfono y email, luego presiona Guardar Cliente.
  • Puedes buscar clientes usando la barra de búsqueda.

2. Gestión de Inventario

  • Agrega nuevos productos definiendo nombre, costo, margen de utilidad y stock inicial.
  • El sistema calcula automáticamente el precio de venta según el margen definido.

3. Órdenes de Trabajo

  • Completa datos de cliente, vehículo y diagnóstico.
  • Agrega repuestos con cantidad y utilidad por unidad.
  • Ingresa el valor de la mano de obra.
  • El sistema calcula total, costo, utilidad y margen automáticamente.
  • Guarda la orden y se generará automáticamente la factura PDF.

4. Facturación

  • Cada orden guardada permite generar una factura PDF.
  • Puedes imprimir o enviar la factura al cliente.
